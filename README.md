# blog-writing-agent

**An AI-powered agent that writes full blog posts from a topic.**

Give it a topic, and the blog-writing-agent takes care of the rest: researching, structuring, and generating a complete, well-written blog post using an LLM-backed multi-step pipeline built with LangChain/LangGraph.
 
---
 
## Features
 
- **Topic-to-blog in one click** — Enter a topic and get a complete, structured blog post
- **Agentic pipeline** — LangGraph orchestrates multi-step reasoning: planning → writing → refining
- **Web UI** — Clean frontend interface to enter topics and view generated output
- **LLM-powered writing** — Leverages large language models via LangChain for high-quality content generation
- **Structured output** — Blog posts with proper headings, sections, and flow
---
 
## Tech Stack
 
| Layer | Technology |
|-------|-----------|
| Agent orchestration | LangGraph |
